# Invest Company

This is a simple static website for the Invest Company project.

## Overview

- Small front-end project with a single-page layout.
- Files included: an HTML entry point, JavaScript, CSS, fonts, and images.

## Project Structure

- index.html — main page
- script.js — site JavaScript
- style.css — site styles
- font/ — custom fonts (if any)
- img/ — images used by the site

## Running Locally

1. Open `index.html` directly in your browser.


## Notes

- This repository appears to be a static site; no build step is required.
- Update `script.js`, `style.css`, and `index.html` to change content or behavior.
